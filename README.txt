موقع معبر (Mabr) — دليل الرفع والتخصيص
==========================================

محتوى المجلد:
- index.html         الصفحة الرئيسية
- article-1.html     أول مقالة ("لستَ وحدك")
- talks.html          صفحة لقاءات (قريبًا)
- contact.html        صفحة تواصل معنا
- style.css           التصميم
- script.js           الترجمة (عربي/إنجليزي) وسلوك الموقع
- README.txt          هذا الملف

------------------------------------------
1) الموقع مربوط بـ GitHub + Netlify
------------------------------------------
أي تعديل تحفظه (commit) على مستودع GitHub (Elianalawi/mabr) في فرع main،
ينشر تلقائيًا على الرابط المباشر خلال دقيقة تقريبًا. تقدر تعدّل مباشرة من
واجهة GitHub بالمتصفح (زر القلم فوق كل ملف)، أو ترفع نسخة محدّثة من الملف
عن طريق "Add file > Upload files".

------------------------------------------
2) تفعيل نموذج التسجيل (النشرة البريدية) في الصفحة الرئيسية
------------------------------------------
1. روح لـ https://formspree.io وسجّل حساب مجاني (حتى 50 رسالة/شهر).
2. أنشئ فورم جديد (New Form)، وخذ الرابط اللي يعطيك (شكله:
   https://formspree.io/f/xxxxxxx)
3. افتح index.html، ابحث عن:
   <form id="signupForm" class="signup-form" action="https://formspree.io/f/YOUR_FORM_ID" ...>
4. استبدل YOUR_FORM_ID برقم الفورم اللي أعطاك Formspree، واحفظ.

بدائل بنفس السهولة: MailerLite أو Kit — عندهم نموذج تسجيل جاهز، ومميزون لأنهم
فعليًا أدوات نشرة بريدية (يعطونك لوحة تحكم لإرسال الأخبار لاحقًا).

------------------------------------------
3) تفعيل فورم "تواصل معنا"
------------------------------------------
نفس خطوات نموذج التسجيل، بس بفورم منفصل (خصص فورم ثاني في حساب Formspree
نفسه أو حساب جديد):
1. أنشئ فورم جديد في Formspree واستخدم بريدك المفضّل لاستقبال الرسائل عليه.
2. افتح contact.html، ابحث عن:
   <form id="contactForm" class="contact-form" action="https://formspree.io/f/YOUR_CONTACT_FORM_ID" ...>
3. استبدل YOUR_CONTACT_FORM_ID برقم الفورم، واحفظ.
كل رسالة يبعتها زائر (الاسم + الرسالة) توصلك مباشرة على بريدك المسجّل هناك.

------------------------------------------
4) تعديل المحتوى
------------------------------------------
- النصوص العربية والإنجليزية كلها موجودة في script.js داخل كائن translations.
  عدّل النص هناك مباشرة (بدون لمس الكود البرمجي حوله).
- نصوص المقالة الطويلة (article-1.html) موجودة مباشرة داخل الصفحة نفسها
  (قسمين: ar-only وen-only).

------------------------------------------
5) وضع الصور (اللوقو وصورة المؤسس)
------------------------------------------
الكود جاهز لعرضهم تلقائيًا بدون أي تعديل كود إضافي:
- الشعار: ضع الملف بجانب index.html باسم logo.png بالضبط.
- صورة المؤسس (في قسم "عنّا"): ضع الملف بجانب index.html باسم founder.jpg بالضبط.
لحد ما تضيف الملفات، تظهر بدائل مؤقتة (نص "معبر" وأيقونة شخص).

------------------------------------------
6) ملاحظات
------------------------------------------
- اللغة الافتراضية عربي (RTL)، وزر "EN" بالأعلى يبدّل للإنجليزية (LTR) ويحفظ اختيار الزائر.
- الموقع متجاوب (يعمل على الموبايل والتابلت).
- لو تحتاج مساعدة في أي خطوة، اطلب مني وأقدر أمشي معك خطوة بخطوة.
